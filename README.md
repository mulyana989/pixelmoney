# PixelWorld 🎮

A simple 2D pixel-based sandbox game built with Phaser 3.

## Features

✨ **Sandbox Building**
- Place and remove blocks
- 6 different block types: Grass, Dirt, Stone, Sand, Water, Wood
- Unlimited creative possibilities

🎮 **Gameplay**
- Smooth player movement
- Real-time world rendering
- Procedurally generated terrain
- Camera following the player

💾 **Save & Load**
- Save your progress locally
- Load previous games
- Clear world function

📦 **Inventory System**
- Track block count
- Quick block selection
- Visual inventory display

## Controls

| Key | Action |
|-----|--------|
| ↑↓←→ or WASD | Move around |
| Mouse Left Click | Place block |
| Mouse Right Click | Remove block |
| 1-6 | Select block type |
| S | Save game |
| L | Load game |
| C | Clear world |

## Block Types

- 🌱 **Grass** - Green grass block
- 🟤 **Dirt** - Brown dirt block
- ⬜ **Stone** - Gray stone block
- 🟨 **Sand** - Yellow sand block
- 💧 **Water** - Blue water block
- 🪵 **Wood** - Brown wood block

## How to Play

1. Use arrow keys or WASD to move around the world
2. Left click to place a block from your inventory
3. Right click to remove a block and add it to inventory
4. Press 1-6 to quickly switch between block types
5. Press S to save your progress
6. Press L to load a saved game

## Installation

```bash
# Clone the repository
git clone https://github.com/mulyana989/pixelmoney.git

# Navigate to the directory
cd pixelmoney

# Start a local server (requires Python or Node.js)
# Using Python 3:
python -m http.server 8000

# Or using Node.js (if installed):
npx http-server

# Open http://localhost:8000 in your browser
```
